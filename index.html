Neighborhood Map

This project is a single page application featuring a map of my hometown with added javascript functionality for highlighted locations and 3rd party data. It takes advantage of jQuery and knockout.JS libraries for features that include closures/event listeners, Model View Controller for separation of concerns, declarative bindings, automatic UI refresh, dependency tracking, and AJAX requests. Google Maps API is utilized to display full screen map, marker placement, marker customization, marker animation, marker infowindows, and map navigation. Wikipedia API utilized to return top articles based on data keywords. Application is dynamic across screen sizes taking advantage of a hidden sidebar triggered by menu button on smaller form factors.
